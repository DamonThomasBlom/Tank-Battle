This asset pack supports all three Unity render pipelines: Built-in, Universal Render Pipeline (URP), and High Definiton Render Pipeline (HDRP).

Textures are shared across all pipelines, but materials are provided separately because each render pipeline uses its own shaders.

Please use the materials from the folder that matches your project's render pipeline.

The Demo Scene includes all Built-in, URP and HDRP materials, it will show the correct material for your project's render pipeline. The scene contains a basic Directional Light for preview purposes. In HDRP projects you may need to adjust the light settings.
